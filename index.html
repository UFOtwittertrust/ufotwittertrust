How to Participate

To assign trust to another Twitter user, tweet using this format (scores from 0 to 100, where 50 is neutral):

Standard command:

#ufotrust @username 80

(assigns 80 trust points - strong trust)

When replying to a user (you can omit their @username in the command):

@someuser That's insightful! #ufotrust 75

(assigns 75 trust points to @someuser)

#ufotrust 50

(in a reply, assigns 50 points - neutral - to the user being replied to)

#ufotrust 20

(in a reply, assigns 20 points - strong distrust - to the user being replied to)

Note: Your trust ratings (0-100) are automatically normalized for influence in the network. 50 is neutral, scores above 50 indicate trust, and scores below 50 indicate distrust.
